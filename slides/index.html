Module 1: Introduction to Reverse Engineering

What is Reverse Engineering?

  • Process of analyzing a system to identify its components and interrelationships, and create representations of the system in another form or at a higher level of abstraction.
  • Understanding how something works without prior knowledge or documentation.

Module 1: Introduction to Reverse Engineering

Why Reverse Engineer?

  • Security Research & Vulnerability Discovery
  • Malware Analysis
  • Interoperability & Compatibility
  • Legacy System Maintenance
  • Competitor Analysis

Module 1: Introduction to Reverse Engineering

Static vs. Dynamic Analysis

  • Static Analysis: Analyzing code without executing it (e.g., examining binary in Ghidra, reading source code).
  • Dynamic Analysis: Analyzing code by executing it and observing its behavior (e.g., debugging, tracing execution, monitoring network traffic).

Module 2: Getting Started with Ghidra

What is Ghidra?

  • A software reverse engineering (SRE) suite of tools developed by the NSA.
  • Free & open source.
  • Features include disassembly, assembly, decompilation, graphing, and scripting.

Module 2: Getting Started with Ghidra

Ghidra's Core Components

  • Project Manager: Manages your reverse engineering projects.
  • Code Browser: The primary interface for analyzing binaries.
  • Decompiler: Converts machine code back into high-level pseudo-code (C-like).
  • Symbol Tree: Lists functions, labels, and other symbols.
  • Data Type Manager: Manages custom data structures and types.

Module 2: Getting Started with Ghidra

Importing a Binary

  1. Open Ghidra Project Manager.
  2. File -> Import File...
  3. Select your binary.
  4. Analyze it with default options for now.

Module 2: Getting Started with Ghidra

Navigating the Code Browser

  • Listing Window: Shows disassembled machine code.
  • Decompiler Window: Shows pseudo-code.
  • Functions Window: Lists all identified functions.
  • Strings Window: Lists all identified strings.
  • Symbol Tree: Browse and manage symbols.
